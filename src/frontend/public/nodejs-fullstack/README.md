# HisabKitab Pro — Node.js Fullstack

Complete self-hosted version. Live Caffeine app is 100% untouched.

## Quick Start

### Backend
```bash
cd nodejs-fullstack/backend
npm install
cp .env.example .env   # fill MySQL credentials
npm run migrate
npm run seed
npm run dev            # http://localhost:3001
```

### Frontend
```bash
cd nodejs-fullstack/frontend
npm install
cp .env.example .env
npm run dev            # http://localhost:5173
```
